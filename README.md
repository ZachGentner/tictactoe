## About The Project

[![Product Name Screen Shot][product-screenshot]](https://www.zachgentner.com/img/tictactoe.png)

This project was part of The Odin Project's Full Stack JavaScript course. Throughout my studies and during development, I learned about building game logic, creating game state, and how to create objects using functions, factories, and modules. This alsoserved as a refresher on navigating and manipulating the DOM.

<!-- USAGE EXAMPLES -->
## Usage

This game is meant to be played on a single device with two players. The players will alternate taking turns placing their tokens until a winner has been determined or a tie has occurred.
